# eslint-config

[Extendable ESLint configuration](http://eslint.org/docs/developer-guide/shareable-configs) used at [Meadow](https://getmeadow.com).

## Usage

```bs
npm install --save-dev @meadow/eslint-config
```

`.eslintrc`:

```json
{
    "extends": "@meadow"
}
```
